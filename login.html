My-Personal-Vudu-Movie-Library

This is a simple service created to list your movies from vudu.com.

Your email and password are not stored or sent anywhere other than to VUDU's servers and are only used to request permission to view your titles. All the resulting data is sent directly from VUDU to your browser.

This is a JavaScript and HTML based application that lives in your browser. You can use the standard features of your modern browsers to view all the source code and network traffic if you're inclined.
